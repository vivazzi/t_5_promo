Задание

- Сверстать страницу по макету https://www.figma.com/file/aFlkfVWh81YGVGusKxIbyb/Тестовое-задание?node-id=0%3A1 и запустить по адресу: /promo/
- Проект разместить в git репозитории
- Для верстки страницы использовать bootstrap 5
- Для запуска слайдера использовать Slick Slider http://kenwheeler.github.io/slick/ (см. Slider Syncing)
- По клику на большую фотографию на слайдере фотографии должны открываться на весь экран и листаться галереей
- Добавить возможность редактировать содержимое слайдера в панели администрирования. Интерфейс редактирования слайдера настроить через admin.py. В списке записей элементов слайдера вывести поля: картинка, название.
- Для загрузки картинок модели слайдера использовать пакет django-filer
- Добавить drag&drop сортировку записей слайдера, используя пакет django-admin-sortable2
- Все кликабельные элементы должны иметь hover эффект
- Форму оставить не рабочей (только верстка). Для input добавить атрибуты: placeholder, required
- Проект запустить на python 3.9, django 4, MySQL
- Все зависимости для запуска проекта расположить в файле req.pip в корне проекта
